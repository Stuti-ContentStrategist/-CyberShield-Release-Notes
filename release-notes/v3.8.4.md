# v3.8.4

**Release Date:** June 15, 2024\
**Audience:** Internal Teams & Developers\
**Type:** Patch Update

***

#### **Summary**

Version 3.8.4 addresses deprecations for legacy systems and resolves key bugs to improve log export reliability, dashboard performance, and endpoint reporting accuracy.

***

#### **Deprecation Warnings**

* &#x20;Legacy FTP-based log exports are deprecated; users must migrate to SFTP or API-based exports.
* Support for 32-bit Linux agents is officially discontinued.
* Older TLS 1.0/1.1 connections are no longer supported; TLS 1.2+ is required for all endpoints.

***

#### **Bug Fixes**

* Fixed rare failures in the log export module when handling large datasets.
* Resolved occasional UI freeze in Security Dashboard filters.
* Corrected mismatch in endpoint version reporting for mixed OS environments.
