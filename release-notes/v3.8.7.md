# v3.8.7

**Release Date:** September 25, 2024\
**Audience:** Internal Teams & Developers\
**Type:** Patch Update

***

#### **Summary**

Version 3.8.7 introduces user-requested enhancements and resolves key bugs to improve dashboard usability, endpoint agent performance, and installation clarity.

***

#### **User Feedback Implemented**

* Dashboard now allows customizable alert thresholds based on user requests.
* Added “Quick Scan” mode to the endpoint agent for faster checks.
* Improved error messages in the installation wizard for better clarity.

***

#### **Bug Fixes**

* Fixed UI alignment issues in the Security Dashboard.
* Resolved rare crashes during bulk endpoint updates.
