# v4.0.8

**Release Date:** February 10, 2025\
**Audience:** Internal Teams & Developers\
**Type:** Patch Update

***

#### Summary

Version 4.0.8 delivers essential security patches addressing authentication and encryption vulnerabilities. This release also includes minor stability fixes to enhance login reliability and threat reporting accuracy.

***

#### Security Advisories

* &#x20;Fixed a vulnerability in the API authentication process that previously allowed potential unauthorized token reuse.
* &#x20;Patched the encryption routine in the endpoint agent to prevent possible data leakage under specific conditions.

***

#### Bug Fixes

* &#x20;Resolved intermittent login failures during periods of high server load.
* &#x20;Corrected inaccurate threat severity labeling in the Security Dashboard for clearer prioritization.
